# 代码生成器 React + ElectronJs

#### 调试界面
```shell
cd react-app
npm start
```

#### 打包界面
```shell
cd react-app
npm build
```

#### 启动软件
```shell
cd electron-js
npm start
```

#### 打包软件
```shell
cd electron-js
npm run-script pack-macos
# https://github.com/electron/electron-packager
# electron-packager . CodeGin --platform=darwin --arch=x64 --out=build --overwrite
```
